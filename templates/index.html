About this Tool

This application demonstrates a classic Caesar cipher implemented in Python. It allows users to encrypt and decrypt text by applying a configurable shift within the alphabet.

The project showcases how a Python backend (Flask) can be integrated with a modern web interface to create an interactive user experience.
