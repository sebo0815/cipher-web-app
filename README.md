# 🔐 Caesar Cipher Web App

This project is a simple web application that demonstrates text encryption and decryption using the Caesar cipher.

It combines a Python backend with a styled web interface to create an interactive encryption tool.

---

## 🚀 Features

* Encrypt and decrypt text
* Adjustable shift value (1–25)
* Real-time interaction via web interface
* Hacker-style UI with animated background

---

## 🛠️ Technologies Used

* Python
* Flask
* HTML, CSS, JavaScript
* Bootstrap

---

## ⚙️ How It Works

The application uses a classic Caesar cipher algorithm.

Each letter in the input text is shifted by a defined number of positions in the alphabet.

Example:

```
HELLO → KHOOR (shift = 3)
```

The backend is implemented in Flask, which handles requests from the frontend and processes the encryption logic.

## 💻 Installation & Usage

Clone the repository:

```
git clone https://github.com/sebo0815/cipher-web-app.git
cd cipher-web-app
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

Open in your browser:

```
http://127.0.0.1:5000
```
